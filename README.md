# STEM Rhythm Game

このリポジトリには、リズムゲームの仕様書に準拠したWebアプリケーションの雛形が含まれています。

## 準備

1. ワークスペースのルートにある `index.html` をブラウザで開きます。ローカルファイルとして開けます。
2. メニューで譜面ファイル、レーンスピード、判定調整を設定し、`START` をクリックします。
3. ゲーム画面が表示され、ノーツが落ちてくるので、`a s d f g h j k l ; ' ]` のキーで入力します。

> **注意**: このバージョンは最小限の動作確認用で、仕様に記載された全機能は未実装です。今後の拡張として、以下を計画しています：
> - 判定幅／スコア計算の細分化
> - スライド・トレース・クリティカル・ダメージノーツの処理
> - HPシステム、ゲームオーバー、特殊演出
> - オプション、メニュー画面のUI改善
> - 譜面セレクターと音声ファイルの自動読み込み

## プロジェクト構成

```
index.html
style.css
src/
  game.js      # ゲームエンジンとレンダラーの参考実装
Charts/        # デモ譜面ファイル（.json/usc）
docs/          # 仕様書
```

## 実装の方針

仕様書に従い、以下のモジュール構成を想定しています。

1. **チャートパーサー**: `.usc` ファイルを読み込み、BPM変更やオブジェクトをタイムライン化する。
2. **ゲームエンジン**: 入力、判定、スコア、HP、コンボ、特殊演出を管理。
3. **レンダラー**: `<canvas>` にレーン、ノーツ、HUD を描画し、レーンスピードやオプションを反映。
4. **入力ハンドラ**: 12レーンのキー入力を受け付け、判定ラインでのタイミングを評価。
5. **UI**: 曲選択、難易度選択、オプション（判定調整A/B、レーンスピード）を提供。

このリポジトリでは、基礎部分をすぐに動かせるように実装してあります。
